# 録音ガイド — 楽器サンプリングのコツ

自分の楽器を Kontakt 音源にするための録音・編集のポイントをまとめたドキュメント。

---

## 1. 環境づくり

### ノイズ対策
- エアコン・換気扇は録音中はオフにする
- スマートフォンは機内モードに（電波ノイズ防止）
- PCのファン音が入る場合はオーディオインターフェースを遠ざける
- 深夜〜早朝が最も環境ノイズが少ない

### 部屋の反響
- 反響が強い部屋（フローリング＋コンクリート壁）は避ける
- 布団・カーペット・本棚で反響を吸収できる
- DI 録音ならルームの影響はゼロ（ベースは DI 推奨）

---

## 2. 録音方法：DI vs マイク

| | DI | マイク |
|---|---|---|
| **向いてる楽器** | ベース、エレキギター | アコギ、ウッドベース、管楽器、打楽器 |
| **ルームノイズ** | 影響なし | 影響あり |
| **セッティング** | 簡単 | マイク位置で音が大きく変わる |
| **音の質感** | クリーンでタイト | 空気感・鳴りが乗る |

### マイク録音のポイント
- **距離**：近いほど低域が強調される（近接効果）。15〜30cm が基準
- **角度**：正面から少しずらすと耳障りな高域が和らぐ
- **複数本**：近接マイク＋ルームマイクで後でブレンド可能

---

## 3. レベル設定

- ピークが **-6dBFS 前後** に収まるよう設定（クリップ厳禁）
- 一番強く弾いたときに -3dBFS を超えないこと
- ノイズフロアは -60dBFS 以下を目標に

---

## 4. マルチサンプリングの進め方

### ノート間隔
- **最小限**：半音3つごと（C, Eb, F#, A...）
- **実用的**：半音2つごと（C, D, E, F#, Ab, Bb...）
- **本格的**：半音1つごと（全音階）

### ベロシティレイヤー
各ノートを強弱別に複数テイク録音する。

| レイヤー | 弾き方 | 目安の強さ |
|---|---|---|
| pp | 最も弱く | 30〜50 |
| mf | 中程度 | 60〜90 |
| ff | 最も強く | 100〜127 |

> **コツ**：毎回同じ強さで弾くのは難しい。DAW のメーターを見ながら、同じピークレベルになるよう意識する。

### テイク数
- 各ノート×各ベロシティで **3〜5テイク** 録っておくと後で選びやすい
- ラウンドロビン（同じノートを複数テイク持つ）があると機械的な繰り返し感が減る

---

## 5. 録音後の編集

### トリミング
- **頭**：アタックの立ち上がりより少し前（数ms）でカット
- **尻尾**：音が自然にフェードアウトするか、フェードアウト処理を加える
- 無音部分（ノイズフロアだけの箇所）は削除

### ループポイント（持続音の場合）
- ループ区間の **始点と終点の波形を合わせる**（ゼロクロス付近で設定）
- ループ境界でクリックノイズが出る場合はクロスフェードを短くかける
- Kontakt の Wave Editor でループ試聴しながら調整

### ファイル名の規則
Kontakt の Auto-map が認識しやすい命名にする：

```
Bass_C2_pp.wav
Bass_C2_mf.wav
Bass_C2_ff.wav
Bass_D2_pp.wav
...
```

またはノート番号で：

```
Bass_036_v1.wav   (036 = MIDI note C2, v1 = velocity layer 1)
Bass_036_v2.wav
Bass_038_v1.wav
```

---

## 6. Kontakt へのインポート

1. Kontakt で **新規インストゥルメント** を作成
2. **Mapping Editor** を開き、サンプルをドラッグ
3. **Auto-map** でノート・ベロシティに自動配置
4. **Wave Editor** でループポイントの微調整・チューニング確認
5. `FretlessBass.ksp` をスクリプトスロットに適用

---

## 7. チェックリスト

- [ ] 環境ノイズを確認（録音だけして波形を目視）
- [ ] レベル設定（ピーク -6dBFS 前後）
- [ ] スマートフォン機内モード

録音中：
- [ ] 各ノートを弦/管の端から端まで網羅
- [ ] ベロシティを意識して一定に弾く
- [ ] テイクごとに少し間を置く（編集しやすくなる）

編集後：
- [ ] 全ファイルのファイル名を統一
- [ ] クリックノイズ・ノイズがないか確認
- [ ] Kontakt でロードして全音域を試奏
